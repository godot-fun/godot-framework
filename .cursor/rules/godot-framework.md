# godot framework Rules

- Before writing business code, check and reuse what already exists in `zfoo/`. Do not rebuild common tools.
- Only change `zfoo/` when you have a clear reason. Put business-specific code in `script/` or `scene/`.
- When looking up framework usage (APIs, setup, examples), read and follow `README.md` in the project root.
